## Используемые технологии
- PostgreSQL 9.6 
- Java 8
- Swing 

Перед запуском, необходимо прописать свои настройки базы данных. По желанию, можно поменять имя БД, которое будет использоваться проектом. 
БД и таблица инициализируется при запуске.
